缠论看盘助手 Skill v2.0
适用场景：对任意股票（A股）进行日线/30F级别的缠论结构分析、买卖点识别与操作计划制定
输入格式：日线截图 + 30F截图（含均线、MACD、成交量）
输出内容：当下定位 + 买卖点判定 + 操作计划 + 风险清单 + 仓位建议

一、输入数据要求
数据项	必需	说明
日线K线图	✅	含MA5/MA10/MA20/MA30、MACD、成交量
30F K线图	✅	含MA5/MA10/MA20/MA30、MACD、成交量
当前股价	✅	盘中实时价格
板块指数	可选	辅助判断系统性风险
关键前高/前低	可选	手工标注的参考点位
二、分析流程（五步法）
第一步：均线系统定位（判断“势”）
均线状态	结论	操作倾向
股价 > MA5 > MA10 > MA20	多头排列	回调做多
股价 < MA5 < MA10 < MA20	空头排列	反弹做空/空仓
均线粘合，股价在中间	震荡无方向	等待方向选择
MA5走平上翘，股价站上MA5	止跌企稳	准备试多
第二步：走势结构划分（判断“位”）
核心逻辑：在日线和30F图上识别“上涨段”和“下跌段”，定位当前处于哪个结构位置。

结构状态	当前定位	对应操作
下跌段末端 + MACD绿柱面积缩小	背驰段	准备买入，等确认信号
下跌段进行中 + 绿柱放大	下跌中继	空仓，不抄底
上涨段末端 + 红柱面积缩小	顶背驰	准备卖出
横盘震荡，无明确方向	中阴阶段	空仓观望
第三步：买卖点判定（三类买卖点）
买点判断逻辑
买卖点	形成条件	确认信号
一买	日线下跌趋势背驰（价格新低，MACD绿柱面积缩小）	30F出现放量底分型 + MACD金叉
二买	一买后反弹，回踩不创新低	30F出现缩量止跌 + 底分型
三买	向上脱离中枢后，回踩不入中枢	股价高于中枢上沿（ZG）
卖点判断逻辑
买卖点	形成条件	确认信号
一卖	日线上涨趋势背驰（价格新高，MACD红柱面积缩小）	30F出现放量顶分型 + MACD死叉
二卖	一卖后反弹，不创新高	30F出现缩量滞涨 + 顶分型
三卖	向下跌破中枢后，反弹不入中枢	股价低于中枢下沿（ZD）
第四步：多级别联立
日线状态	30F状态	综合结论
空头趋势	下跌末端（背驰）	大级别空头+小级别买点 → 只做短线反弹，轻仓、快进快出
空头趋势	上涨启动	大级别空头+小级别上涨 → 可参与反弹，但严格止损
多头趋势	回调末端	大级别多头+小级别买点 → 可重仓参与
震荡无方向	任意	无趋势 → 空仓或极轻仓
第五步：制定操作计划
必须包含三要素：

买入条件（触发才能动手）

价格区间

K线形态（底分型/放量阳线）

MACD状态（金叉/绿柱缩短）

成交量状态（放量/缩量）

止损位（跌破必须走）

通常设在前低或买入K线最低点下方

目标位（分批离场）

第一目标：MA5/MA10

第二目标：前高/中枢上沿

时间止损：持仓N天未到目标则离场

三、优化项1：入场信号优先级排序
当多个买点信号同时出现时，按以下优先级执行：

优先级	信号类型	信号强度	仓位建议	说明
P0（最高）	日线一买 + 30F二买共振	⭐⭐⭐⭐⭐	3-4成	大小级别同时确认，胜率最高
P1（高）	日线一买确认	⭐⭐⭐⭐	2-3成	大级别买点，可重仓参与
P2（中）	30F一买 + 日线盘整背驰	⭐⭐⭐	1-2成	小级别买点，轻仓试错
P3（低）	30F二买（日线仍空头）	⭐⭐	1成	仅做短线反弹，快进快出
P4（极低）	无背驰的任意底分型	⭐	0成	不参与，等待确认
实战应用：

如果出现 P0 或 P1 信号 → 积极准备，可在确认后第一时间进场

如果出现 P2 或 P3 信号 → 轻仓试错，严格止损，不恋战

如果只有 P4 信号 → 忽略，继续空仓

四、优化项2：持仓期间的每日跟踪清单
买入后，每天按以下清单检查持仓状态：

每日开盘前（09:00-09:20）
检查项	操作
查看公告	是否有重大利好/利空公告
查看隔夜外盘/板块	大盘和板块是否安全
设置条件单	止损单、止盈单提前挂好
每日盘中（09:30-15:00）
时间	盯什么	信号判断
09:30-10:00	开盘30分钟K线方向	放量阳线 → 持仓；放量阴线 → 警惕
10:30	30F MACD是否死叉/金叉	死叉 → 考虑减仓；金叉 → 持有
13:00-13:30	下午开盘方向	延续上午趋势则持有；反向则警惕
14:45-14:55	尾盘是否守住关键位	守住 → 持有；跌破 → 减仓/离场
每日收盘后（15:00-16:00）
检查项	操作
评估当日K线形态	收阳/收阴/长上影/长下影
评估MACD状态	红柱扩大/缩小？绿柱扩大/缩小？
评估成交量	放量/缩量/平量
更新止损位	如果股价上涨，上移止损位（保本或保护利润）
更新目标位	如果接近目标，分批止盈
持仓期间的三个“离场信号”
信号	操作
破位止损	跌破预设止损位 → 无条件离场
时间止损	持仓达到预设周期（如5天）仍未到目标 → 离场
顶背驰信号	30F出现顶背驰（MACD红柱面积缩小+顶分型）→ 离场
五、优化项3：不同大盘环境下的仓位系数调整
根据大盘（上证指数）状态，对计划仓位进行系数调整：

大盘状态分类
大盘状态	定义	仓位系数
强势	上证指数 > MA20，且MA20向上	100%（按原计划执行）
中性	上证指数在MA20附近震荡	60%（原计划×0.6）
弱势	上证指数 < MA20，且MA20向下	30%（原计划×0.3）
极端弱势	上证指数单日跌幅 > 2% 或连续3日下跌	0%（暂停所有买入）
实战示例
原计划仓位	大盘状态	调整后仓位	操作
2成	中性（系数0.6）	1.2成	减半执行，更谨慎
3成	弱势（系数0.3）	0.9成	极轻仓试错
1成	极端弱势（系数0）	0成	放弃本次交易
大盘与个股的四种关系
大盘	个股	结论
涨	涨	✅ 最佳：可重仓
涨	跌	⚠️ 个股弱：关注补涨机会，但不抄底
跌	涨	⚠️ 个股强：可轻仓参与，但需设紧止损
跌	跌	❌ 最差：空仓观望，不操作
