# 缠论理论定理落地改进措施

目标：把可计算的缠论结构尽量标记到 K 线上，并用分层显示辅助校验，最终提高三类买点和三类卖点识别准确性。

## 已落地

| 理论点 | 程序对象 | 图层/输出 | 用途 |
| --- | --- | --- | --- |
| 笔定理 | `fractals`、`strokes` | 分型、笔 | 所有高级结构的基础 |
| 线段破坏 | `segments.status`、`theory_marks.segment_break` | 线段、理论标记 | 判断旧线段确认结束、新线段开始 |
| 走势中枢 | `centers` | 中枢 ZD/ZG 虚线 | 趋势、盘整、三买三卖判断基础 |
| 中枢形成/延伸 | `theory_marks.center_formed/center_extend` | 理论标记 | 区分中枢刚形成、继续震荡或可能升级 |
| 中枢离开/回试 | `theory_marks.center_leave/center_retest` | 理论标记 | 三买/三卖前置条件可视化 |
| 趋势/盘整 | `trend`、`theory_marks.trend_state` | 理论标记、右侧状态 | 判断一买/一卖必须处于趋势背驰 |
| 背驰 | `divergences` | 背驰图层 | 一买/一卖的核心触发条件 |
| MACD 零轴 | `theory_marks.macd_zero` | 理论标记 | 辅助一买/二买、一卖/二卖过滤 |
| 三类买卖点 | `signals` | 买卖点图层 | 最终交易信号候选、确认、失效 |

## 下一步优先级

| 优先级 | 改进项 | 规则依据 | 实现要点 |
| --- | --- | --- | --- |
| P0 | 完善线段特征序列破坏 | 线段分解定理、线段破坏充要条件 | 当前已实现状态机和三笔重叠基础，后续要把“特征序列分型”独立建模并显示 |
| P0 | 买卖点依赖链可解释化 | 背驰-买卖点定理、买卖点完备性定理 | 每个 B1/B2/B3/S1/S2/S3 记录引用的中枢、离开段、回试段、背驰段 |
| P1 | 二买/二卖次级别确认 | 买卖点定律一、MACD 定律 | 二买/二卖不能只看本级别回踩，需要引入低一级别一买/一卖或等价候选 |
| P1 | 走势级别递归 | 走势分解定理、级别延续定理 | 支持日线/30分钟/5分钟联动，把大级别买卖点收缩到小级别背驰区间 |
| P1 | 中枢升级提示 | 中枢中心定理、级别延续定理 | 当相邻中枢或围绕两个中枢的波动区间重叠，标记“疑似高级别中枢” |
| P2 | 均线吻规则 | 飞吻/唇吻/湿吻 | 可做辅助图层，不直接替代三类买卖点 |
| P2 | 背驰力度多算法对比 | 趋势力度、平均力度 | 同时输出价格幅度、MACD 面积、单位时间力度，供过滤配置 |

## 图层设计

- 分型：顶/底基础结构。
- 笔：由交替顶底分型构成。
- 线段：确认线段实线，运行线段虚线。
- 中枢：ZD/ZG 两条虚线，并显示价格。
- 背驰：趋势背驰/盘整背驰单独显示。
- 理论标记：中枢形成、延伸、离开、回试、趋势/盘整、MACD 零轴。
- 买卖点：只显示最终 B1/B2/B3/S1/S2/S3 候选、确认、失效。

## 买卖点准确性原则

1. 一买/一卖必须来自趋势背驰，且趋势至少包含两个同向不重叠中枢。
2. 二买/二卖必须依赖已确认的一买/一卖，且需要次级别结构确认。
3. 三买/三卖必须来自离开中枢后的回试不进中枢。
4. 当下未完成结构只标为候选或理论标记，不直接当作确认买卖点。
5. 大级别买卖点应能在次级别背驰段中进一步缩小范围。
